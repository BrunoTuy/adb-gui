# adb-gui

Graphical interface for executing adb commands

## Getting Started

The application is under development and may present some intabilities. To run it is necessary to download the fonts.

### Prerequisites

NodeJS and npm

### Run application

Download the modules

```
npm install
```

Launch application
```
npm start
```
